How to Start Developing

```
npm install
export PORT=5000
// tab1 - run mongo for both testing and data storage
sudo mongod --config /usr/local/etc/mongod.conf 
// tab 2 - start node.js server
npm start 
// tab 3 - test watch
npm run tw 
```

How to Deploy
